> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scaffold a static Hugo site at `/Users/benedictsteele/Development/profound.fyi` using the `hugo-paper` theme installed as a Hugo Module, ready to deploy on Netlify.

**Architecture:** Hugo (extended) builds the site. The theme is consumed as a Hugo Module via Go modules — no submodules, no vendored copy. Netlify reads `netlify.toml` and runs `hugo --gc --minify` on every push. Content lives under `content/`; theme assets live under `_modules/` cache and are not tracked.

- Go 1.25.6 (installed at `/Users/benedictsteele/homebrew/bin/go`)
